#Lab drizzle-orm + bun.

    Este é um laboratório simples para testar e entender a camada de persistência de dados em uma aplicação usando drizzle-ORM e typescript.
    O objetivo não é construir um sistema completo, mas sim entender o ciclo de vida da conexão com o banco de dados, e entender como gerenciadores de pacotes mais atualizados como bun, funcionam e se comportam em um projeto.

Objetivos do laboratório:
    - Entender a configuração do ambiente desde o 0 com bun.
    - Aprender a configurar o drizzle-ORM do zero, e entender o que cada configuração faz.
    - Entender como conectar o drizzle ao postgreSQL usando string de conexão.
    Aplicar os primeiros padrões de clean architecture em uma aplicação, como divisão em camadas (repository, service, DTO, controller).
